My Public GitHub Projects

A collection of scientific software and utilities primarily developed in LabVIEW and JavaScript.

  • yappari-5-1

    "Yet Another Program for Analysis and Research in Impedance" (YAPPARI). A comprehensive Windows application for impedance spectroscopy analysis. Features include non-linear parametric fitting, Distribution of Relaxation Times (DRT) analysis, Hilbert transform (Z-Hit) calculations, and spectra simulations.

    Language: Labview

  • Keithley622x Control

    A state machine-based application built in Labview 2018 to control a Keithley 6221 AC/DC current source. It offers partial compatibility with the Keithley 6220 model, enabling automated control for precision experiments.

    Language: Labview

  • PPMS Control

    LabVIEW VIs for controlling a Quantum Design Physical Property Measurement System (PPMS). Includes modules for setting and monitoring temperature and magnetic fields via GPIB.

    Language: Labview

  • Van der Pauw Calculator

    A simple VI (Virtual Instrument) in Labview 2018 designed to calculate resistivity and Hall coefficient values from measurements obtained using the van der Pauw method.

    Language: Labview

  • High Pressure Calculator

    A utility for high-pressure physics experiments. This Labview 2018 program provides a simple calculation of pressure based on the spectral shift of the ruby fluorescence line.

    Language: Labview

  • Seebeck Calculator

    A LabVIEW program to calculate the absolute values of the Seebeck coefficient for individual thermoelectric legs and for complete thermoelectric couples.

    Language: Labview

  • Teaching Simulations

    A collection of interactive web-based simulations for educational purposes, designed to help visualize and understand various scientific concepts.

    Language: JS

  • Programmable Resistor

    A LabVIEW project to design and simulate the behavior of a programmable resistor, useful for circuit design and automated testing environments.

    Language: Labview

  • QR10x LabVIEW Drivers

    A set of LabVIEW drivers for controlling QR-10x programmable resistors, enabling integration into automated filtering in Wheatstone bridge systems.

    Language: LabVIEW

  • SPS Control

    LabVIEW application for controlling a Spark Plasma Sintering (SPS) system. It allows for the creation and execution of temperature and pressure cycles.

    Language: LabVIEW

  • PVD Control

    This LabVIEW project is designed to control a Physical Vapor Deposition (PVD) system, managing the deposition process parameters.

    Language: LabVIEW

  • CVD Control

    A LabVIEW application for controlling a Chemical Vapor Deposition (CVD) system, enabling the automation of gas flow, temperature, and pressure for thin film deposition.

    Language: LabVIEW
